# Avatar

Pills are used to indicate a temporary state or show higlighted values.
Avatar are useful to show the initials of a name or the picture of a user.
By default the color will be randomly generated from the name, but there are also 3 different variants `primary`, `secondary` and `tertiary`.
If you add a `url` will be used as image and it is also compatible with [unavatar](https://unavatar.io/)
