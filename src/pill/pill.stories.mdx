# Pill

Pills are used to indicate a temporary state or show higlighted values.
There are 3 different variants `primary`, `secondary` and `tertiary` or the one you decide to create, but also other 4 intents `info`, `success`, `warning` and `error`.

## How to use this component

### Angular

```js
import { Pill } from '@papanasi/angular';
import '@papanasi/angular/dist/papanasi.css';

@NgModule({
  declarations: [Pill]
})
export class AppModule {}
```

```html
<div className="app">
  <pill variant="primary">Primary</pill>
</div>
```

### React

```js
import { Pill } from '@papanasi/react';
import '@papanasi/react/dist/papanasi.css';
export default function App() {
  return <Pill variant="primary">Primary</Pill>;
}
```

### Vue

```js
<script lang="ts">
import { Pill } from "@papanasi/vue";
import "@papanasi/vue/dist/papanasi.css";
export default {
  name: "App",
  components: {
    Pill,
  },
};
</script>
<template>
  <Pill variant="primary">Primary</Pill>
</template>
```

### Web Components

```js
import { Pill } from '@papanasi/webcomponents';
import '@papanasi/webcomponents/dist/papanasi.css';
customElements.get('pa-button') || customElements.define('pa-button', Pill);
```

```html
<pa-button variant="primary">Primary</pa-button>
```
