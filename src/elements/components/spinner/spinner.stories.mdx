# Spinner

This Spinner should cover most of the scenarios where you need to show a simple loading.
By Spinner the color will be `basic`, but there are also 3 different variants `primary`, `secondary` and `tertiary`.
You can also force to be `full` or `fullscreen` spinner if you don't want it inline.
